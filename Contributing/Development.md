# Development Guide

While this project's contents are delivered in the form of a SaltStack
formula[^1] and _should_ work in that broader context, the contents have only
specifically been tested within the context of the watchmaker project[^2]. As
such, see _that_ project's Development Guide documentation[^3] for a sketch of
project setup. Also see that project's "Testing New Formulas"[^4] guidance.

[^1]: See the ["vendor" documentation](https://docs.saltproject.io/en/latest/topics/development/conventions/formulas.html)
[^2]: See the [watchmaker project home](https://github.com/plus3it/watchmaker)
[^3]: See: [Development Guide documentation](https://watchmaker.readthedocs.io/en/stable/contributing.html#development-guide)
[^4]: See: [Testing New Formulas](https://watchmaker.readthedocs.io/en/stable/customization/NewFormulas.html)
